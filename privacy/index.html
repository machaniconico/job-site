Privacy

プライバシーポリシー

回答データの取り扱い

本サイトの診断はすべて、あなたのブラウザ内（端末上）で完結します。設問への回答や診断結果は、サーバーへ送信されません。途中保存や結果表示のために、回答内容をお使いのブラウザの localStorage に保存します。これは同じブラウザでの再開や再表示のためだけに使われ、外部に共有されることはありません。

保存データの削除

保存された回答は、診断画面の「回答をリセット」または結果画面の「もう一度」で消去できます。ブラウザの設定からサイトデータ（localStorage）を削除することでも消去できます。

外部送信・解析・広告

本サンプル実装では、アクセス解析・広告・問い合わせフォーム等による個人情報の収集は行っていません。「結果をXでシェア」を押した場合のみ、あなたの操作によって X（旧Twitter）の投稿画面へ要約テキストが渡されます（自動送信ではありません）。将来、解析や広告を導入する場合は、本ポリシーを更新します。

第三者サービス

本サイトは GitHub Pages 上で配信しています。配信に伴い、GitHub 社のインフラ側でアクセスログが記録される場合があります。詳細は GitHub のプライバシー方針をご確認ください。

お問い合わせ

本ポリシーに関するご質問は、サイト管理者までお寄せください。
